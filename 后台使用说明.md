# 可视化后台使用说明

## 入口

部署后打开：

```text
https://asd715.github.io/eesdfresume/admin/
```

## 第一次登录

这个后台使用 Sveltia CMS，内容会直接保存到 GitHub 仓库 `asd715/eesdfresume`。

第一次进入后台时，按页面提示使用 GitHub 授权。若页面要求填写 token，请在 GitHub 创建一个 Fine-grained personal access token：

- Repository access：只选择 `asd715/eesdfresume`
- Permissions：Contents 选择 `Read and write`

保存后，后台即可编辑 `content/site.json`。

## 能编辑什么

- 个人信息：姓名、标题、邮箱、抖音、头像
- 首页核心数据卡片
- 商业实践案例
- 项目档案馆
- 正在研究什么
- 我的思考栏目
- 证据墙图片、标题和说明

## 修改后如何生效

后台点击保存后，会把修改提交到 GitHub。GitHub Pages 通常几十秒到几分钟后自动更新公开网站。

公开网站：

```text
https://asd715.github.io/eesdfresume/
```
